# Netflix Content Analysis 

## Overview
This repository contains an analysis of Netflix movies and TV shows using Microsoft Excel. The project focuses on exploring content trends, ratings, genres, and release patterns to generate meaningful insights through data visualization and dashboard creation.

---

##  Problem Statement
Netflix hosts thousands of movies and TV shows, making it difficult to identify patterns in content distribution, ratings, genres, and release trends. This project aims to analyze Netflix content data and uncover insights that can support better understanding of audience preferences and platform growth.

---

## Goal of the Analysis
The primary goal of this analysis was to clean and explore the Netflix dataset, identify key trends in movies and TV shows, and build an interactive Excel dashboard to visualize important insights related to content performance and distribution.

---

##  Dataset
The dataset contains Netflix content information including:
- Title
- Type
- Description
- Release Year
- Runtime
- Genres
- Production Countries
- Seasons
- IMDb Score
- IMDb Votes
- TMDb Popularity
- TMDb Score

---

##  Tools Used
- Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Slicers
  - Conditional Formatting
  - Data Cleaning Functions

---

## Key Questions Answered
- What is the distribution of Movies vs TV Shows on Netflix?
- Which genres are most common on Netflix?
- How has Netflix content changed over the years?
- What relationship exists between IMDb ratings and popularity?
- Which content types have higher average ratings?

---

##  Key Findings
- Movies make up the majority of Netflix content compared to TV shows.
- Drama and Comedy are among the most frequently appearing genres.
- Netflix content releases increased significantly after 2015.
- Titles with higher IMDb scores generally showed higher TMDb popularity.
- TV shows tend to have longer overall runtimes due to multiple seasons.

---

##  Recommendations
- Expand investment in highly rated genres such as Drama and Comedy.
- Continue increasing original content production to match growing audience demand.
- Use rating and popularity insights to improve recommendation strategies.
- Conduct deeper audience analysis to identify trends in viewer preferences.
